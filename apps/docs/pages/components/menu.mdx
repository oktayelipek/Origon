# Menu

Anchored action list. Portals to `document.body` so it escapes overflow
clips. Source: Figma `- Menu` (`12:55798`).

## Live preview

## API

<PropsTable rows={[
  { name: 'trigger', type: 'ReactElement', description: 'Element that toggles the menu. Cloned to intercept clicks + set aria attrs.' },
  { name: 'items', type: 'MenuItem[]', description: 'Menu items. Each has value, label, optional icon/disabled/danger/divider.' },
  { name: 'onSelect', type: '(value: string) => void', description: 'Called when a non-disabled item is clicked.' },
  { name: 'side', type: '"bottom-start" | "bottom-end" | "top-start" | "top-end"', defaultValue: '"bottom-start"' },
]} />

## Behavior

- Click outside or `Escape` closes.
- Portals to body, clamps to viewport with 4px margin.
- `divider: true` on an item renders a horizontal rule *before* it.
- `danger: true` uses red text (destructive intent).
