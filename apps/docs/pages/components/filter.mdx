# Filter

Compact pill buttons used above data tables and list views. Source: Figma
`Header Filter`, `Header Filter [Icons]`, `Sort Filter` (page `12:84954`).

Two variants:
- **Filter** — labelled dropdown trigger with an optional dismiss button.
- **SortFilter** — column sort toggle (asc / desc / unsorted).

Dropdown behavior itself is provided by the [Dropdown](/components/dropdown)
component; Filter is just the trigger surface.

      <Filter label="Type" onClick={() => alert('open picker')} />
      <SortFilter label="Volume" direction={dir} onClick={() => setDir(dir === 'asc' ? 'desc' : 'asc')} />

    </div>
  );
