# Range Slider

Single- or dual-thumb slider. Pass `value={n}` for single, `value={[a, b]}`
for dual. Source: Figma `- Range Selector` (`12:54109`).

## Live preview

<Demo />

export const Demo = () => {
  const [single, setSingle] = useState(35);
  const [range, setRange] = useState([20, 80]);
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: 32, padding: 32, background: 'var(--semantic-level-basement, #0b0f1a)', borderRadius: 12, marginBottom: 24, maxWidth: 480 }}>
      <RangeSlider label={`Single thumb: ${single}%`} value={single} onChange={setSingle} showLabels />
      <RangeSlider label={`Dual thumb: ${range[0]} — ${range[1]}`} value={range} onChange={setRange} showLabels />
      <RangeSlider label="Disabled" value={40} onChange={() => {}} disabled />
    </div>
  );
};

## Behavior

- Track click moves the nearest thumb.
- Pointer events (mouse/touch) for drag.
- Each thumb has `role="slider"` + `aria-valuemin/max/now/valuetext`.
- Arrow key handling is on the v4 roadmap.
