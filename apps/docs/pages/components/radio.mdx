# Radio

Single-choice input, always used inside a `<RadioGroup>` for state
management + accessibility. Source: Figma `FK-Radio` (`12:54256`).

## Accessibility

- RadioGroup wraps with `role="radiogroup"` and forwards `aria-label`.
- Each Radio is a native `<input type="radio">` — Tab lands on the group,
  Arrow keys move between options (native behavior when `name` is shared).
- `<label>` wraps the input so clicking the label toggles selection.
