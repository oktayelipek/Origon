# Icons + Logos

Origon UI ships two icon layers:

1. **Starter set** (`Icon` component, 12 glyphs) — for the design system's
   own docs and quick prototyping.
2. **Full library** (158 icons under "Essentials" in Figma) — auto-exported
   from Figma via `scripts/export-icons.mjs`. Each icon ships as its own
   named React component under `@origon/react/icons`.

Additional multi-color brand assets (bank logos, crypto marks) ship separately
via `@origon/react/logos` — see the [Logos page](/components/logos).

## Full Figma library

**158 icons** synced from the Figma Essentials group. Each ships as its own
React component (tree-shakable) plus a name-keyed map for dynamic use.

<FigmaIconGrid />

export const FigmaIconGrid = () => {
  const [query, setQuery] = useState('');
  const filtered = GENERATED_ICON_NAMES.filter((n) => n.toLowerCase().includes(query.toLowerCase()));
  return (
    <div style={{ marginBottom: 24 }}>
      <input
        placeholder={`Search ${GENERATED_ICON_NAMES.length} icons…`}
        value={query}
        onChange={(e) => setQuery(e.target.value)}
        style={{
          display: 'block',
          width: '100%',
          padding: '10px 14px',
          marginBottom: 16,
          fontSize: 14,
          fontFamily: 'inherit',
          background: 'var(--semantic-level-elevation, #131f2f)',
          color: 'var(--semantic-text-focus, #f0f4f7)',
          border: '1px solid var(--semantic-border-level-4, #192f43)',
          borderRadius: 8,
          outline: 'none',
        }}
      />
      <div style={{ fontSize: 12, opacity: 0.6, marginBottom: 12 }}>
        {filtered.length} of {GENERATED_ICON_NAMES.length}
      </div>
      <div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(96px, 1fr))', gap: 8, padding: 16, background: 'var(--semantic-level-basement, #0b0f1a)', borderRadius: 12, color: '#f0f4f7', maxHeight: 480, overflowY: 'auto' }}>
        {filtered.map((name) => {
          const IconComp = ICON_COMPONENT_MAP[name];
          return (
            <div
              key={name}
              title={name}
              style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', gap: 4, padding: 8, background: 'var(--semantic-level-elevation, #131f2f)', borderRadius: 6, cursor: 'pointer', overflow: 'hidden' }}
            >
              <IconComp size={20} />
              <code style={{ fontSize: 9, opacity: 0.6, textAlign: 'center', maxWidth: '100%', textOverflow: 'ellipsis', overflow: 'hidden', whiteSpace: 'nowrap' }}>{name}</code>
            </div>
          );
        })}
      </div>
    </div>
  );
};

```tsx
// Named import from the full library:
import { StarIcon, CheckIcon } from '@origon/react/icons';

<StarIcon size={20} style={{ color: 'gold' }} />

// Or the starter set (for smaller bundle when only 12 glyphs needed):
import { Icon } from '@origon/react';

<Icon name="star" size={20} style={{ color: 'gold' }} />

// Dynamic lookup:
import { ICON_COMPONENT_MAP } from '@origon/react/icons';
const IconComp = ICON_COMPONENT_MAP['star'];
<IconComp size={20} />
```

## Generated components (`@origon/react/icons`)

Each icon is a React component: `<NameIcon />` — e.g., `<StarIcon />`,
`<CheckIcon />`, `<ArrowDownIcon />`. All accept:

## Regenerating

The generator is idempotent — re-run to pull the latest from Figma:

```bash
FIGMA_TOKEN=figd_… node scripts/export-icons.mjs
```

Optional `--group=Essentials|Others|…` to export a different section.
