# Input

Text input. Source: Figma `{FKInput}` (`12:68514`) — 81 variants across many
axes. This v1 covers the core Default feature; specialized features
(Numeric, Phone, ID, Encrypted PIN, Multi-line, Extensions) will ship as
separate components.

### PasswordInput API

<PropsTable rows={[
  { name: 'showStrengthMeter', type: 'boolean', defaultValue: 'false', description: 'Renders the 3-bar strength gauge below the field.' },

Extends all `Input` props except `type` (fixed to password) and `rightIcon` (replaced by the visibility toggle).

<PropsTable rows={[

## States

- **Inactive** — resting; bg `blueGray.200`.
- **Focus / Typing** — active; bg lightens to `blueGray.400`.
- **Done** — filled but blurred (visually same as Inactive currently).
- **Error** — 1px red border (`red.600`) + red error message.
- **Disabled** — non-interactive; muted text color.

## Accessibility

- Native `<input>` — full keyboard, focus, screen reader support.
- Label uses `<label htmlFor>` for click-to-focus and screen reader association.
- Error text is linked via `aria-describedby` and sets `aria-invalid="true"`.
