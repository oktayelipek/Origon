# Dropdown / Combo

Anchored select. Setting `combobox` turns it into a filterable combobox.
Source: Figma `- Dropdown/Combo` (`12:85331`).

## Live preview

<Demo />

export const Demo = () => {
  const [pair, setPair] = useState('BTCTRY');
  const [country, setCountry] = useState('');
  const pairs = [
    { value: 'BTCTRY', label: 'BTC / TRY' },
    { value: 'ETHTRY', label: 'ETH / TRY' },
    { value: 'BNBTRY', label: 'BNB / TRY' },
    { value: 'AVAXTRY', label: 'AVAX / TRY' },
    { value: 'SOLTRY', label: 'SOL / TRY' },
  ];
  const countries = [
    { value: 'tr', label: 'Türkiye' },
    { value: 'de', label: 'Deutschland' },
    { value: 'fr', label: 'France' },
    { value: 'us', label: 'United States' },
    { value: 'jp', label: 'Japan' },
    { value: 'br', label: 'Brasil' },
  ];
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: 20, padding: 24, background: 'var(--semantic-level-basement, #0b0f1a)', borderRadius: 12, marginBottom: 24, maxWidth: 400 }}>
      <Dropdown options={pairs} value={pair} onChange={setPair} placeholder="Choose a pair" />
      <Dropdown options={countries} value={country} onChange={setCountry} placeholder="Choose a country" combobox />
      <Dropdown options={pairs} placeholder="Disabled" disabled />
    </div>
  );
};

## Behavior

- Click outside or press `Escape` to close.
- Trigger is a `<button aria-haspopup="listbox">`; popover uses `role="listbox"` with `role="option"` children.
- Currently does not support keyboard navigation over options (arrow keys) — that arrives in v2.
