# Notification Badge

Dot or count overlay for notifications, unread counts, alerts. Wraps any
child and positions itself at the top-right corner. Source: Figma
`- Notification Badge` (`12:55791`).

## Live preview

<div style={{ display: 'flex', gap: 32, padding: 24, background: 'var(--semantic-level-basement, #0b0f1a)', borderRadius: 12, marginBottom: 24, alignItems: 'center' }}>
  <NotificationBadge count={3}><Button variant="outline" iconPosition="only" icon={<Icon name="star" />} aria-label="Favorites" /></NotificationBadge>
  <NotificationBadge count={12} tone="brand"><Button variant="outline">Inbox</Button></NotificationBadge>
  <NotificationBadge count={128} max={99} tone="warning"><Button variant="outline">Orders</Button></NotificationBadge>
  <NotificationBadge dot tone="success"><Button variant="outline">Online</Button></NotificationBadge>
  <NotificationBadge count={0} showZero><Button variant="outline">All caught up</Button></NotificationBadge>
</div>

## API

<PropsTable rows={[
