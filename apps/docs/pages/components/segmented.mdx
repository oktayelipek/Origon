# Segmented Control

Mutually-exclusive choice group. Common uses: chart timespans, sort orders,
view modes. Distinct from `Tabs` — always renders as a rounded pill.
Source: Figma `- Segmented Control` (`12:49453`).

<PropsTable rows={[

Rendered as `role="radiogroup"` with each option `role="radio"` + `aria-checked`.
