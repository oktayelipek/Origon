## Accessibility

- Native `<input type="checkbox">` — full keyboard support, focus indicator, screen reader announcement.
- Label uses `<label htmlFor>` — clicking the label toggles the input.
- `error` state also sets `aria-invalid="true"` so assistive tech announces the validation state.
