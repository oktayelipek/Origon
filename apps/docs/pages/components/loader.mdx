# Loader

Indeterminate loading indicator. Source: Figma component set `FK-Loader`
(node `12:58010`). Rendered as a 12-petal SVG that pulses each petal in
sequence (~1s cycle), producing a wave motion.

## Accessibility

- Rendered as `role="progressbar"` with `aria-label` (default: "Loading").
- `aria-live="polite"` — screen readers announce presence but don't interrupt.
- Prefers-reduced-motion is **not** yet respected; a static frame will be shown
  in a future revision.

## Do / don't

- ✅ Use `size="small"` inside buttons and rows.
- ✅ Use `size="large"` for full-view loading states.
- ❌ Don't use for determinate progress — use a Progress component instead
  (coming later).
- ❌ Don't wrap in another animation — the wave motion is deliberate.
