# Chart

Line / area / bar / candlestick chart with hover crosshair + tooltip.
For heavier viz (brushing, log scale, multi-axis, WebGL) integrate a chart
library and pass Origon tokens as theme values.

Source: Figma `- Charts` (`12:86299`).

## Usage

<Tabs items={['React', 'Flutter', 'Swift']}>
<Tabs.Tab>

```tsx
import { Chart } from '@origon/react';

<Chart
  kind="area"
  series={[{ data: prices }]}
  width={400}
  height={200}
/>
```

</Tabs.Tab>
<Tabs.Tab>

```dart
import 'package:origon_ui/origon_ui.dart';

OrigonChart(
  kind: OrigonChartKind.area,
  series: [OrigonChartSeries(data: prices)],
  width: 400,
  height: 200,
);
```

</Tabs.Tab>
<Tabs.Tab>

```swift
import OrigonUI

OrigonChart(
    kind: .area,
    series: [OrigonChartSeries(data: prices)],
    width: 400,
    height: 200
)
```

</Tabs.Tab>
</Tabs>
