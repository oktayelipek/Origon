# Tabs

Horizontal navigation between related views. Two variants:
- **`underline`** — subtle bottom border for content pane headers.
- **`pill`** — rounded pill for toolbar-style switching.

Source: Figma `{Tab}` (`12:48473`).

## Accessibility

- Container: `role="tablist"` + optional `aria-label`.
- Each tab: `role="tab"` + `aria-selected`.
- Native `<button>` — full keyboard focus + activation.
