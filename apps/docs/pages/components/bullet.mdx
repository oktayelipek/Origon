# Bullet

Step / pagination indicator. Source: Figma `Bullet [Line]` (`8:837`) and
`Bullet [Dots]` (`8:866`). Two visual variants, arbitrary count, one active
index. Typically used for onboarding steps, carousels, and paged wizards.

## Anatomy

- **Container** — horizontal flex row, gap `spacing.xs` (8px).
- **Item** — a pill (radius `xxl`): 40×4 for line, 8×8 for dots.
- **Colors** — active = `semantic.text.focus`, inactive = `colors.blueGray.300`.

## Accessibility

- Rendered as `role="tablist"` with `aria-label` describing progress (`Step N of M`).
- The active item carries `aria-current="step"`.
- Bullet is decorative-only when count/active reflect other primary UI —
  screen reader users typically already know their position from the current
  screen's context.
