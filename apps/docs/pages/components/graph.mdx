# Graph

Compact sparkline for row and card contexts. Distinct from
[Chart](/components/chart) — no axes or grid; single series only; auto-
colored by trend direction (green up, red down). Source: Figma
`- Graph` (`12:84746`).

## Usage — inline with values

<div style={{ display: 'flex', flexDirection: 'column', gap: 8, padding: 24, background: 'var(--semantic-level-basement, #0b0f1a)', borderRadius: 12, marginBottom: 24, color: '#f0f4f7' }}>
  {[
    { pair: 'BTC/TRY', change: '+2.4%', trend: 'up', data: [10, 12, 11, 15, 14, 18, 20] },
    { pair: 'ETH/TRY', change: '-1.1%', trend: 'down', data: [22, 20, 24, 18, 15, 13, 10] },
    { pair: 'SOL/TRY', change: '+5.3%', trend: 'up', data: [8, 9, 12, 15, 18, 22, 26] },
  ].map((row) => (
    <div key={row.pair} style={{ display: 'grid', gridTemplateColumns: '80px 1fr 60px', alignItems: 'center', gap: 12, padding: '8px 12px', background: 'var(--semantic-level-elevation, #131f2f)', borderRadius: 8 }}>
      <span style={{ fontFamily: 'monospace' }}>{row.pair}</span>
      <Graph data={row.data} width={120} height={30} />
      <span style={{ textAlign: 'right', color: row.trend === 'up' ? '#4fa963' : '#dc3323' }}>{row.change}</span>
    </div>
  ))}
</div>
