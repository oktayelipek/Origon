# Chip

Compact rounded label. Two roles:

- **Display** — static pill for tags, categories, metadata. Line or solid.
- **Selector** — interactive toggle for period pickers, filter values.

Source: Figma `Chip` (`12:86248`) and `Chip [Selector]` (`12:86235`).

// Display

## Anatomy

- Corner radius `xxl` (28) — pill shape at all sizes.
- Padding scales with size: X-Small uses `spacing.xs / xxs`, Large uses `spacing.4xl / md`.
- Selector uses `semantic.button.primary` for selected background,
  `colors.blueGray[300]` outline for unselected.

## Accessibility

- Display chips render as `<span>` — decorative.
- Selector chips render as `<button role="switch" aria-checked>` — keyboard
  focusable, announced with state.
